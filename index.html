Sobre Mí

Soy Técnico Superior en Ciencia de Datos con enfoque en soluciones reales para empresas. Mi trabajo se centra en tomar datos dispersos, limpiarlos, analizarlos y presentarlos de forma clara para que las decisiones de negocio se basen en evidencia concreta.

Cada proyecto que desarrollo es un sistema completo: desde la extracción de datos hasta el dashboard final, pasando por ETL, análisis estadístico y visualización interactiva. Todo pensado para que el usuario final pueda operar sin conocimientos técnicos.
