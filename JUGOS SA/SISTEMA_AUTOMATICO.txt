AL INICIAR, EL SISTEMA DETECTA:

[1] PYTHON DISPONIBLE:
    - Python del sistema (C:\Python311, C:\Python39, etc.)
    - Python en el PATH
    - WinPython portable en la carpeta
    - Arquitectura de cada uno (32 o 64 bits)

[2] DRIVERS ODBC:
    - Drivers x64 (*.accdb)
    - Drivers x32 (*.mdb)
    - Compatibilidad con cada Python

[3] MEJOR COMBINACIÓN:
    - Encuentra Python + ODBC compatible
    - Si hay varios, elige el mejor
    - Si no hay compatible, usa modo CSV

[4] MODO DE OPERACIÓN:
    - Modo ACCESS: Lee directamente desde Access
    - Modo CSV: Usa archivos CSV existentes
