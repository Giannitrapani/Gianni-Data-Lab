ARCHIVOS IMPORTANTES
================================================================================

PARA EJECUTAR:
   INICIAR_SISTEMA.bat    - Inicia el sistema completo
   DETENER_SERVICIO.bat   - Detiene actualizaciones automáticas

PARA DIAGNOSTICAR:
   DIAGNOSTICO.bat        - Detecta problemas automáticamente

PARA CONFIGURAR:
   CONFIGURAR_RUTA_RED.bat           - Si Access está en red
   CONFIGURAR_PYTHON_PORTABLE.bat    - Si necesitas Python portable

   INSTRUCCIONES.txt      - Manual completo del sistema
   GUIA_SIN_ADMIN.txt    - Guía para usuarios sin permisos admin

================================================================================
¿QUÉ HACE ESTE SISTEMA?
================================================================================

- Lee datos desde una base de datos Access (.accdb)
- Genera dashboards interactivos con gráficos
- Se actualiza automáticamente cada 1 hora
- Funciona sin conexión a internet
- Portable: se puede ejecutar desde pendrive o red


================================================================================
CARACTERÍSTICAS TÉCNICAS
================================================================================

✓ Actualización automática cada hora
✓ Compatible con ODBC x32 y x64
✓ Funciona sin permisos de administrador
✓ Portable (pendrive, red, local)
✓ Detección automática de problemas
✓ Sistema de caché inteligente
✓ Compatible con Windows 7, 8, 10, 11

PREGUNTAS FRECUENTES
================================================================================

P: ¿Necesito internet?
R: NO. Todo funciona offline.

P: ¿Necesito permisos de administrador?
R: NO. Con Python Portable funciona sin admin.

P: ¿Puedo usarlo en un pendrive?
R: SÍ. Copia toda la carpeta y ejecuta.

P: ¿Funciona en red?
R: SÍ. Puedes colocarlo en carpeta compartida.

P: ¿Modifica la base de datos?
R: NO. Solo lee, nunca escribe.

P: ¿Se actualiza automáticamente?
R: SÍ. Cada 1 hora sin intervención manual.
