🍊JUGOS S.A.

En producción

Sistema integral de gestión y análisis de envases (bins) para empresa de producción de jugos. Incluye dashboard interactivo con KPIs en tiempo real, sistema semáforo de proveedores, análisis logístico y presentación ejecutiva para gerencia. Desarrollado completamente con Python, Plotly y HTML.

📊 Dashboard Completo KPIs en tiempo real, stock de bins, semáforo de proveedores y análisis logístico en 3 pestañas interactivas. Abrir dashboard → 🎬 Presentación Gerencial Presentación interactiva de 19 slides con navegación estilo Prezi. Pensada para reuniones de directorio. Ver presentación →

Stack Tecnológico

Python 3.x Pandas NumPy Plotly pyodbc Microsoft Access ODBC HTML / CSS / JS Windows Batch Jupyter Notebooks

Características del Sistema

  • Dashboard interactivo con 3 módulos: Stock Global, Proveedores, Logística
  • KPIs con umbrales de alerta: crítico (5.000 bins), advertencia (8.000 bins)
  • Sistema semáforo de proveedores (verde/amarillo/rojo)
  • Análisis logístico: temporal, por distancia, por especie y por productor
  • Actualización automática cada hora desde base de datos Access
  • Funciona sin instalación, sin admin, sin internet (portable)
  • Compatible con Windows 7, 8, 10 y 11
  • Presentación ejecutiva de 19 slides con navegación interactiva
  • Detección automática de drivers ODBC (32 y 64 bits)
  • Sistema de diagnóstico y verificación integrado
