🍊JUGOS S.A.

En produccion

Sistema integral de gestion y analisis de envases (bins) para empresa de produccion de jugos. Incluye dashboard interactivo con KPIs en tiempo real, sistema semaforo de proveedores, analisis logistico y presentacion ejecutiva para gerencia. Desarrollado completamente con Python, Plotly y HTML.

📊 Dashboard Completo KPIs en tiempo real, stock de bins, semaforo de proveedores y analisis logistico en 3 pestanas interactivas. Abrir dashboard → 🎬 Presentacion Gerencial Presentacion interactiva de 19 slides con navegacion estilo Prezi. Pensada para reuniones de directorio. Ver presentacion →

Stack Tecnologico

Python 3.x Pandas NumPy Plotly pyodbc Microsoft Access ODBC HTML / CSS / JS Windows Batch Jupyter Notebooks

Caracteristicas del Sistema

  • Dashboard interactivo con 3 modulos: Stock Global, Proveedores, Logistica
  • KPIs con umbrales de alerta: critico (5.000 bins), advertencia (8.000 bins)
  • Sistema semaforo de proveedores (verde/amarillo/rojo)
  • Analisis logistico: temporal, por distancia, por especie y por productor
  • Actualizacion automatica cada hora desde base de datos Access
  • Funciona sin instalacion, sin admin, sin internet (portable)
  • Compatible con Windows 7, 8, 10 y 11
  • Presentacion ejecutiva de 19 slides con navegacion interactiva
  • Deteccion automatica de drivers ODBC (32 y 64 bits)
  • Sistema de diagnostico y verificacion integrado
